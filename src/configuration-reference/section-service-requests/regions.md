# Regions

**Page Type:** Table (URL: [https://dad3.ndrn.org/admin/admindisplayregions](https://dad3.ndrn.org/admin/admindisplayregions)) <!--style: Subtitle -->

***Associated Record Types:** [Counties](../section-clients/counties.md) (Clients), [Users](../section-users/index.md) (Users)* <!-- style: Subtitle -->

 <!-- style: Subtitle -->

Regions can be used to create geographical groups of counties and/or to associate staff with one or more geographical areas. Region information can then be pulled using the Report Generator.

# New Features of DAD 3.0 Administration

<!-- AUTO-SUMMARY-START -->

## [New Form Experience](new-form-experience.md)

<p>In DAD 3.0 Administration settings, associations between different types of fields are integrated directly into the New/Edit forms for each item. Each associated field is displayed as an additional form tab.</p>

## [Admin Settings Export](settings-export.md)

<p>Settings pages that contain lists of items can now be exported to Excel. You must view the list in Grid format to display the “Export to Excel” button.</p>

```admonish important
Fields you wish to export must be displayed as individual columns. Before exporting, select the menu icon for any column header, expand the **Column** submenu, and select all desired fields for display (you may need to scroll down to see all field options).
```

## [Field Display Editor](field-display-editor.md)

<p>The Field Display Editor is an Administration Tool that allows agencies to fully customize the layout of display order and layout of fields for Clients, Service Requests and Projects to align with their workflow and use of DAD.</p>

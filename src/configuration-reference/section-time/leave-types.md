# Leave Types

**Page Type:** Table (URL: [https://dad3.ndrn.org/admin/admindisplayadvleavetypes](https://dad3.ndrn.org/admin/admindisplayadvleavetypes)) <!--style: Subtitle -->

Leave Types are categories used to group [Leave options](leave.md). A typical use is to categorize leave into Paid and Unpaid types. The **Leave Type** dropdown is displayed on the Record Time form when the **Leave** Timekeeper Type is selected. The dropdown options for **Leave** are those with the selected **Leave Type**.
